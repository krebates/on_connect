# OnConnect

Gem that helps connect OnConnect api

## Installation

Add this line to your application's Gemfile:

    gem 'on_connect'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install on_connect

## Usage

on_connect::Movie.search('Frozen')
